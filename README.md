# Sunrin Today

Sunrin Internet High School Lunch Service Scheduler

<br>

## Installation

```sh
git clone https://github.com/sunrin-today/scheduler
```

## Build

```sh
docker build . -t ghcr.io/sunrin-today/scheduler
```

## Start

```sh
docker run ghcr.io/sunrin-today/scheduler
```
